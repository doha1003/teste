✅ 테스트 완료

모든 dh- 접두사 클래스가 기존 클래스와 동일한 스타일을 적용하고 있는지 확인하세요.

  • dh-c-* : Component (버튼, 카드 등)
  • dh-l-* : Layout (컨테이너, 그리드 등)
  • dh-f-* : Fortune (운세 관련)
  • dh-t-* : Test (테스트 관련)
  • dh-u-* : Utility (유틸리티)
