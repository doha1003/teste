🛠️ 404 오류 발생 시 해결 방법

만약 실제로 404 오류가 발생한다면 다음 순서로 해결하세요:

  1. 1
    브라우저 캐시 클리어
    Ctrl + Shift + R (하드 리프레시) 또는 개발자도구에서 캐시 비활성화
  2. 2
    로컬 서버 재시작
    python -m http.server 3000 또는 npm run dev
  3. 3
    CSS 번들 재생성
    npm run build:css로 dist/styles.min.css 다시 생성
  4. 4
    개발자도구 확인
    F12 → Network 탭에서 실패한 요청의 정확한 경로 확인
