🔍 디버깅 체크리스트

  1. 페이지 로드: 각 링크를 클릭하여 페이지가 로드되는지 확인
  2. 콘솔 오류: F12를 눌러 개발자 도구에서 Console 탭 확인
  3. 시작 버튼:"테스트 시작하기" 버튼이 표시되는지 확인
  4. 버튼 클릭: 시작 버튼을 클릭했을 때 반응하는지 확인
  5. 첫 질문: 질문이 표시되고 선택 옵션이 나타나는지 확인
